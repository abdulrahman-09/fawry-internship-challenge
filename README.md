# Fawry Internship Challenge

A simple Java console application that simulates a traffic radar system.

The radar processes vehicle observations, checks them against traffic rules, and creates fines when violations are found.

## Features

- Supports multiple vehicle types: private cars, trucks, and buses.
- Detects speed limit violations.
- Detects missing seatbelt violations.
- Calculates total fine amounts.
- Tracks violation counts by rule.

## Requirements

- Java 21
